üìù Usage Example

The new formatters are easy to use throughout the codebase:

import { formatNumber, formatDate, formatTime, formatGlucoseNumber } from '../utils/formatting/formatters';

// Numbers automatically use correct separators
const hba1c = formatNumber(6.8, 1);  // "6.8" in EN, "6,8" in DE

// Dates use correct format
const date = formatDate(new Date(2023, 11, 31));  // "12/31/2023" in EN, "31.12.2023" in DE

// Times use 12h or 24h format
const time = formatTime(new Date(2023, 11, 31, 14, 30));  // "2:30 PM" in EN, "14:30" in DE

// Glucose values
const glucose = formatGlucoseNumber(5.5, 1);  // "5.5" in EN, "5,5" in DE
